## Technologies Used

- **Frontend**: React, TailwindCSS, shadcn/ui  
- **Backend**: Supabase (PostgreSQL, Auth, Storage)  
- **AI Integration**: Google Gemini API  
- **State Management**: TanStack Query (React Query)  
- **Forms and Validation**: React Hook Form, Zod  
- **Routing**: React Router  
- **Deployment**: Vercel

## Challenges and Learnings

Key challenges included synchronizing complex data structures across user, group, and goal contexts while keeping the UX seamless. Implementing a secure and scalable AI integration using Gemini required thoughtful prompt engineering and efficient API usage. Group collaboration features also introduced complexity in permission handling and data visibility, which was successfully solved through role-based access and scoped queries.

## Outcome

G15 is actively used as a comprehensive financial management tool for individuals and small teams. The app provides meaningful insights, motivates users with gamified features, and promotes better financial habits. The scalable architecture also lays the foundation for future enhancements, including deeper AI integrations and expanded collaboration tools.

## Setup Instructions

1. Clone the repository  
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Set up environment variables in a `.env` file:
   - Supabase project URL and public API key
   - Gemini API key for AI assistant
4. Initialize Supabase:
   - Create tables using the provided SQL schemas
   - Set up auth and deploy edge functions
5. Start the development server:
   ```bash
   npm run dev
   ```

## Database Schema

Key tables in the PostgreSQL database:

- `profiles`: User data  
- `goals`: Financial goals  
- `transactions`: Expenses and income  
- `budget_categories`: Budget planning  
- `badges` / `user_badges`: Gamification system  
- `groups` / `group_members`: Financial groups  
- `showcase`: Public achievements  
- `subscriptions`: Pro plan data
